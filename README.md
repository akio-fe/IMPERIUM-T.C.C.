# Projeto IMPERIUM 🚀 
**Trabalho de Conclusão de Curso (TCC) - Etec Jardim Ângela**

O **IMPERIUM** é uma plataforma de e-commerce de vestuário de alto desempenho, desenvolvida como projeto final para o curso Técnico em Desenvolvimento de Sistemas. O diferencial do sistema é a experiência imersiva do usuário, utilizando visualizações 3D de produtos e uma arquitetura de back-end focada em segurança e escalabilidade.

## 🛠️ Tecnologias Utilizadas

- **Back-end:** PHP (Lógica de negócio e integração)
- **Banco de Dados:** MySQL (Persistência de dados) e Firebase (Dados em tempo real/Autenticação)
- **Front-end:** JavaScript, Three.js (Visualização 3D)
- **Integrações:** API do Mercado Pago (Processamento de pagamentos)
- **DevOps/Segurança:** Git, GitHub Secrets, Branching Workflow

## ✨ Principais Funcionalidades

- **Catálogo 3D:** Visualização interativa dos produtos para uma melhor experiência de compra.
- **Pagamento Seguro:** Integração total com o Mercado Pago para transações criptografadas.
- **Gestão de Usuários:** Sistema de login e perfis integrados ao Firebase.
- **Painel Administrativo:** Interface para controle de estoque e pedidos.

## ⚙️ Diferenciais de Engenharia

Durante o desenvolvimento, foram aplicadas boas práticas de engenharia de software:
- **Git Flow:** Gerenciamento de funcionalidades através de branches organizadas.
- **Segurança:** Uso de variáveis de ambiente e secret scanning para proteção de chaves de API.
- **Modelagem de Dados:** Estrutura SQL otimizada para garantir a integridade das informações de vendas.
